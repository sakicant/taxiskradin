# Content Guidelines

- Use headers to create scannable structure (H2 for sections, H3 for subsections)
- Include a clear introduction that states the value proposition
- Break long content into digestible sections
- Use bullet points for lists of 3+ items
- Include calls-to-action where appropriate
- Cite sources for data claims
- Optimize for readability: short paragraphs, clear transitions
