# taxiskradin.hr (TAXI Antonio, Skradin)

Sister site of taxisibenik.hr. Same owner, same driver, same design and build
system. This file carries the conventions and knowledge from the taxisibenik
project (C:\Users\sakic\taxisibenik-code) so any session here starts informed.

## What this site is

- Private taxi and transfer service for Skradin and its surroundings, run by
  one driver: Antonio Sakic. One car: a Skoda Superb, up to 4 passengers.
- 11 languages: en (site root), hr, de, pl, cs, it, fr, nl, sl, hu, sk under
  their own prefixes with localized slugs.
- Skradin-origin content lives HERE; Sibenik-origin content lives on
  taxisibenik.hr. The two sites complement each other and cross-link; they
  must never compete for the same keywords. ACI Marina Skradin content was
  moved here from taxisibenik for that reason.

## Build workflow (identical to taxisibenik)

- Pages live in `src/pages/<page-id>/<lang>/{meta.json,content.html}`;
  shared markup in `src/partials/` (per-language header/footer/widget).
- Run `python build.py` to generate the site. NEVER hand-edit generated
  `index.html` files; they are overwritten on every build.
- `PRICES` in `script.js` is the single source of truth for fares. The
  booking widget dropdown is driven by the `GROUPS` list in `script.js`.
- `build.py` computes an asset version hash for `styles.css` + `script.js`
  cache busting, and writes `sitemap.xml`.
- Local preview: `.claude/launch.json` defines a static server. PHP pages
  (booking backend, admin) cannot run in the static preview; preview them
  via published artifacts or test on the host.

## Standing rules (owner's preferences, non-negotiable)

- NO em dashes ever, in any content, any language.
- The car is always written "Skoda Superb" (with the diacritic in content:
  "Škoda Superb"). Never a different model.
- Antonio speaks ONLY Croatian and English. Never claim he speaks German or
  any other language, in any translation.
- Mobile-friendliness is top priority: check every change at mobile width,
  not just desktop.
- Antonio sells TRANSPORT only. He is not a tour agency: no experiences, no
  per-person pricing, no charging for advice or tastings. Wording on pages
  that touch tours/wineries must make this explicit.
- Prices are fixed per car (up to 4 passengers), agreed in advance. A 20%
  advance (minimum 20 EUR) confirms a booking; balance on the day, cash or
  card. Bookings need at least 2 hours notice.
- Working hours (updated 2026-07-15): Mon-Fri 07:00-14:00 and 16:00-21:00,
  Sat-Sun 06:00-16:00. Available 24/7 for rides booked in advance. Keep the
  footer, home page and LocalBusiness openingHoursSpecification in sync.
- Git: descriptive commits explaining why; end every commit message with
  the trailer: Co-Authored-By: Claude Opus 4.8 <noreply@anthropic.com>
  Remind the owner about uncommitted changes; commit/push after confirming.
- `config.php` (real DB + admin credentials) is gitignored and lives only on
  the server. `config.sample.php` is the template.

## Backend (PHP + MySQL, mirrors taxisibenik)

- `booking-submit.php` stores bookings and emails the owner + customer.
- `offers-api.php` serves special offers; the admin manages them.
- Admin: on taxisibenik the admin folder was renamed to a secret path
  (`manage-k29q7x/`) and gated with HTTP Basic Auth (salted SHA-256 in
  config). THIS PROJECT still has the old open `admin/` folder; rename it to
  its own secret path (pick a DIFFERENT secret than taxisibenik's) and port
  the auth gate before going live. See docs/port-from-taxisibenik.md.
- Deployment target: cPanel hosting (mydataknox). DB + user created in
  cPanel, tables imported via phpMyAdmin, config.php created on the server.

## SEO knowledge that carries over

- Route pages: one page per direction, fixed fare shown, `taxi-<from>-to-<to>`
  slug pattern (localized per language). Generated by scripts (see
  `scripts/`), driven by a master route list + the PRICES matrix.
- Rich snippets reality: Google largely does not show FAQ/review stars for
  this site type; Google Business Profile is the real lever for stars.
- Every page gets hreflang alternates; sitemap should carry xhtml:link
  alternates (port the build.py update).
- llms.txt + robots.txt kept current and readable; robots must NOT reveal
  the secret admin path.
