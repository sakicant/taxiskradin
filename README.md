# taxiskradin.hr

Website for **TAXI Antonio – Skradin** (taxiskradin.hr). Same design system,
build pipeline and TAXI Antonio branding as taxisibenik.hr.

Static site generated by `build.py` from `src/pages/<page-id>/<lang>/`
(`meta.json` + `content.html`) plus the shared partials in `src/partials/`.
English is served at the root; the other languages under their own prefix
(`/hr/`, `/de/`, ...). Run `python build.py` after editing any page or partial.
